# 2026-05-22 工作记录

## Daily Booking Dashboard 修复

### Bug 修复：ETD + Lane 筛选后数据消失
- **现象**: 选 ETD 范围 + All LANES 后，Summary 显示 "No data matches the selected filters"
- **根因**: `cascadeFilters()` 函数中 DEL 字段名错误 `r.del` → `r.del_port`，导致 ETD/Lane 筛选后数据全部消失
- **修复**: `r.del` → `r.del_port`

### Bug 修复：选 Trunk Lane 后数据完全不更新
- **现象**: 设置 ETD 范围后再选 Trunk Lane，无论选哪个 lane，数据不更新（仍显示旧结果或 0）
- **根因**: `LANE_ETD_MAP` 未从 JSON payload 加载（漏写 `LANE_ETD_MAP = payload.lane_etd_map || {}`），导致 `onLaneChange` 中访问 `LANE_ETD_MAP[lane]` 抛出 TypeError，函数崩溃，`cascadeFilters()` 和 `refreshAll()` 从未执行
- **修复**: 在 payload 加载块中增加 `LANE_ETD_MAP = payload.lane_etd_map || {};`
