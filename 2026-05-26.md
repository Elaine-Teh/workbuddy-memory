### Bug 修复：TEU 数据不正确
- **现象**: BZCF2621W 显示 TEU = 6,896，但期望约 4,680
- **根因**: Excel `Booking TTL Teu` 列 (col AX, index 49) 实际存的是 **Booking Count**（票数），不是真正的 TEU 箱型换算值
  - 全量 BZCF2621W: Booking TTL Teu = 6,886 = Booking Count
  - 但 ft20=984, ft40=2,953, ft40rf=6 → 真正 TEU = 6,902
- **修复**: 不再读取 Excel 的 `Booking TTL Teu` 列，改用公式直接计算：
  - `TEU = ft20*1 + ft40*2 + ft40rf*2`
- **结果**: 
  - BZCF2621W 筛选后 TEU = 4,690（原来是 4,680=Booking Count）
  - 全站 Total TEU = 223,448（原来 221,927=Booking Count）
  - TEU 和 Booking Count 现在正确分离
